  1. What do Null deviance and residual deviance of summary output mean?

Deviance is a measure of goodness of fit of the model. the higher the deviance worse the results are. The Null deviance shows how well the response variable is predicted by intercept only model (a model that includes only the intercep). The Residual deviance shows how well the response variable is predicted with the inclusion of predictors. From the summary we can see that the addition of 1 (332-331=1, look at degrees of freedom) independent variable (for the model_class_ba) decreased deviance from 441.48 to 431.91. So the residual deviance has decreased by 9.57 (441.48-431.91) with the loss of 1 degree of freadom. We can say that model with ba included better explains the data, than intercept only model.

  1. Is it possible to calculate R square for logistic regression?

For logistic regression we use McFaddens pseudo-R squared .

R2McFadden=1-log(Lc)/log(Lnull)

Here Lc shows the max. likelihood value for the current fitted model, and Lnull is for the corresponding value but for the intercept only model. Thus, when our model is predicting all of the variation in the outcome, Lc is 1, sequentially log(Lc) will be 0 and R2McFadden=1.Closer the value of R2_McFadden is to 1 better is our model.Here R2_McFadden is 0.7782182. We can conclude that our model explains the total variation by 77.8%.

intercept_only_model <- glm(class~1,data = Bio_final, family="binomial")

  2. What is the difference between the ROC and the precision-recall curve (PRC)? When do we need to consider PRC? Why? Plot the PRC if it is applicable to your models?

ROC curves should be used when the number of observations are equal for each class. Precision-Recall curves can be used when data is inbalanced. As ROC curves is applicable for datasets with imbalanced classes, when the proportion of positive to negative instances changes in a test set, the ROC curves stays the same.
